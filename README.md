# NHSRwaitinglistsimulator

This repository is a work in progress.

The aim of the repository is to provide the functionality to simulate waiting lists under user provided assumptions. 

Inputs: tbc



Outputs: 
    - Single experiment - a list of clock starts and clock stops
    - Multiple experiment - a daily description of the waiting list with mean queue size, 95% confidence interval and 1st and 3rd quartiles.



To run an example: tbc
